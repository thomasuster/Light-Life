between()function
public function between(min:Number, max:Number, exclusive:Boolean = false):Matcher

Matches if the values is between the min and max values.

Parameters

min:Number — Minimum value
 
max:Number — Maximum value
 
exclusive:Boolean (default = false) — true to allow the value to be equal to the min or max, false to require the value to be inside the range of the min and max.

Returns
Matcher — Matcher

Example 
      assertThat(2, between(2, 4));
      // passes
           assertThat(2, between(2, 4, true));
      // fails
