Packageorg.hamcrest.core
Classpublic class GivenMatcher
InheritanceGivenMatcher BaseMatcher Object

Conditionally matches a Matcher. Used in coordination with AnyOfMatcher and AllOfMatcher to express conditional logic as a short-circuit functional style conditional expression. Syntax shortcut (vs EvaluateMatcher) for expressing simple conditional matcher logic. If the specified condition is true, matches the specified Matcher. Else, it either matches or does not match depending on the optional otherwise parameter.
